UnitAI is a next-generation measurement companion designed to bridge the gap between abstract data and human intuition. Unlike traditional converters that simply swap digits, UnitAI uses Google Gemini Flash Lite to provide instant, real-world context for every measurement you encounter.

### Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
